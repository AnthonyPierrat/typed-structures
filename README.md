<h2 align="center">Typed Structures</h2>

Typed Structures is an MIT-licensed on-going open source project meant to bring a fullset of data structures to the awesome TypeScript ðŸŽ‰

<h3>Current features</h3>

 - [x] Singly linked list
 - [x] Doubly linked list
 - [x] Map
 - [x] Set
 - [x] Stack
 - [x] Queue
 - [x] Binary tree
 - [x] Buffer
 - [x] Data query engine
 
## How to install
```bash
npm install typed-structures
```
  
## How to import
```javascript
import { Map } from 'typed-structures';
let map: Map<number, number> = new Map<number, number>();
map.put(1, 1);
map.containsKey(1);
// true
```
